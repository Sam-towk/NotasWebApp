# 📄 Aplicativo de Notas em Markdown

Um sistema web simples e eficiente para criação, listagem e exclusão de notas locais estruturadas em arquivos físicos `.md` (Markdown), desenvolvido com Laravel e Tailwind CSS.

## 🚀 Tecnologias Utilizadas

* **Backend:** Laravel 11+
* **Frontend:** Blade Components & Tailwind CSS (via Laravel Breeze)
* **Storage:** Sistema de arquivos local do PHP/Laravel
